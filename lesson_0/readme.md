## 📌 Что такое C++ и зачем он нужен?

Начнём с базы: разберёмся, что вообще представляет собой этот язык.

**C++** - это высокоуровневый объектно-ориентированный язык программирования, разработанный **Бьёрном Страуструпом** в **1983 году**. Его синтаксис на 99% похож на синтаксис языка-предшественника - **C**, однако в C++ добавлена поддержка классов, объектов и других возможностей ООП.

Поскольку C++ работает напрямую с железом и операционной системой, он обладает высокой скоростью и гибкостью. Именно поэтому он используется для создания:

- **Операционных систем** (например, Windows 10/11)
- **Игровых движков** (Unity, Unreal Engine, Source, Source 2 и множество других AAA-игр)
- **Систем управления базами данных** (например, MySQL)
- **Браузеров** (Firefox, Chrome)
- **Тяжёлого ПО** (Adobe Photoshop и другие)
- **Приложений реального времени** (финтех, робототехника, телекоммуникации)

C++ - один из самых быстрых языков высокого уровня: по производительности он близок к ассемблеру, но при этом предоставляет удобные абстракции.

Важная особенность: C++ позволяет **вручную управлять памятью**, чего не допускают большинство современных языков. Это делает его очень мощным, но вместе с тем и более сложным в написании. Появляется риск **утечки памяти** - ситуации, когда программист выделяет память под переменные, объекты или массивы, но не освобождает её. Также требуется понимание понятий **кучи** и **стека** - подробнее об этом поговорим в следующих уроках.

## 🛠️ Установка необходимых инструментов

Теперь, когда мы разобрались с теорией, перейдём к установке нужных программ.

Так как я использую Linux, здесь нет привычного для многих громоздкого Visual Studio. Но зато есть **VS Code** - лёгкий и удобный редактор кода. Для его установки перехожу на официальный сайт:

 [code.visualstudio.com](https://code.visualstudio.com/)

Скачиваем версию `.deb` (для Linux), затем открываем терминал сочетанием клавиш `Ctrl + Alt + T` или `Win + Enter`, переходим в папку загрузок:

```bash
cd ~/Downloads
```

И вводим команду установки файла:

```bash
sudo apt install ./имя_файла_vscode_amd64.deb
```

После ввода пароля VS Code установится.

Затем мы создадим папку проекта и откроем её в редакторе.

### 🔌 Необходимые расширения

Чтобы запускать код на C++ прямо из VS Code, нужно установить два плагина:

1. **C/C++ Extension Pack** - официальный пакет от Microsoft
2. **C/C++ Compile Run** от `danielpinto8zz6`

> Если вы используете Visual Studio (не VS Code), устанавливать расширения не нужно - поддержка C++ встроена.

### Подключение библиотеки

Любая программа на C++ начинается с подключения заголовочных файлов:

```cpp
#include <iostream>
```

Разберём эту строку подробно:

| Элемент | Значение |
|---|---|
| `#` | Маркер директивы препроцессора. Строки с `#` - это инструкции для препроцессора (части компилятора), они обрабатываются до синтаксического анализа кода. |
| `include` | Директива включения файла. Препроцессор находит указанный файл и буквально вставляет его содержимое в исходный код на место этой строки. |
| `<iostream>` | Заголовочный файл стандартной библиотеки ввода-вывода. Угловые скобки `< >` говорят компилятору искать файл в системных путях. |

> Важно помнить: компьютер читает код сверху вниз, поэтому подключение библиотек всегда идёт первым.

### Функция main и вывод в консоль

```cpp
int main()
{
    std::cout << "Hello, I use Linux btw!";
    return 0;
}
```

Разберём каждый элемент:

| Элемент                     | Значение                                                                                                                                                        |     |
| --------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------- | --- |
| `int`                       | Тип возвращаемого значения функции. Стандарт C++ требует, чтобы функция `main` возвращала `int`.                                                                |     |
| `main`                      | Зарезервированное имя точки входа программы. Без неё компилятор выдаст ошибку: «точка входа не найдена».                                                        |     |
| `()`                        | Список передаваемых параметров. В данном случае он пустой.                                                                                                      |     |
| `{`                         | Открывающая фигурная скобка - начало тела функции.                                                                                                              |     |
| `std::`                     | Квалификатор пространства имён. Указывает компилятору, что мы используем объект из стандартной библиотеки C++.                                                  |     |
| `cout`                      | Глобальный объект потока вывода. Можно расшифровать как **c**onsole **out**put. Без `std::` работать не будет (есть альтернативы - разберём в следующем уроке). |     |
| `<<`                        | Оператор вставки. Передаёт значение справа в поток вывода слева.                                                                                                |     |
| `"Hello, I use Linux btw!"` | Строковый литерал. Вместо него можно передать переменную, символ, число и так далее - об этом в следующем уроке.                                                |     |
| `;`                         | Конец инструкции. Обязателен в конце каждой строки кода - без него компилятор выдаст синтаксическую ошибку.                                                     |     |
| `return 0`                  | Оператор возврата. Прерывает выполнение функции и возвращает код завершения `0`, что означает «программа завершилась успешно».                                  |     |
| `}`                         | Закрывающая фигурная скобка - конец тела функции.                                                                                                               |     |
